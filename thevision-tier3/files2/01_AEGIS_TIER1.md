# AEGIS — Tier 1
## AI-Driven SPARTA Emulation & Gap Identification System

---

## What It Is

AEGIS is an autonomous adversary simulation platform built
specifically for Haven-1. It generates AI-driven attack chains
mapped to SPARTA v3.2, executes them against a digital twin
of Haven-1's architecture, identifies every missing security
control, and generates the actual fixes.

**One command runs the full pipeline in under 10 minutes.**

## The Problem It Solves

Haven-1 is not a satellite. It is a crewed vehicle where
a successful cyberattack does not mean data loss.
It means crew members cannot breathe, cannot communicate,
or cannot come home.

Traditional security:
- Point-in-time pen test twice per year
- Findings sit in a backlog
- No automated remediation
- No crew safety weighting

AEGIS:
- Continuous automated red team
- Every finding has a deployed fix
- Ranked by crew safety impact
- Runs in 8.5 seconds

## Technical Architecture

```
SPARTA v3.2 Knowledge Base (12 Haven-1 techniques)
              │
              ▼
AI Mission Planner (Claude API)
  Reads scenario YAML
  Reasons over SPARTA KB + Haven-1 architecture
  Generates SPARTA-mapped attack chain
              │
              ▼ chain.json
Go Execution Runner
  Executes each step against Docker lab
  6 containers: ground + space segment
  Logs pass/fail + crew safety impact
              │
              ▼ results.json
AI Gap Analyzer (Claude API)
  Analyzes each successful step
  Identifies missing SPARTA countermeasures
  Maps to NIST 800-53
  Scores by crew safety weight
              │
              ▼ gap_report.json
AI Debrief Writer (Claude API)
  Writes full engagement report
  Plain English for leadership
  Technical depth for engineers
              │
              ▼ debrief.md + heatmap.html
```
